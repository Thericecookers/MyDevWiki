* git add *:修改提交到缓冲区  
* git commit -m "**some message**"将缓冲区提交到本地分支  
* git push:将本地分支提交到远程仓库  
* git branch:查看分支(也可以直接添加分支，git branch **branch_name**)  
    -M 重命名分支  
    -r 远程分支
